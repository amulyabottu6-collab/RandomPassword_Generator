# 🔐 Random Password Generator – Python Project
## 📌 Project Overview
This is a simple Python-based Random Password Generator that creates strong and secure passwords based on user preferences such as length, letters, numbers, and symbols.
---
## 🚀 Features
- Generates strong random passwords
- User can choose password length
- Option to include:
  - Letters (A–Z, a–z)
  - Numbers (0–9)
  - Symbols (!@#$%^&*)
- Ensures at least one character type is selected
- Simple command-line interface
---
## 🛠️ Technologies Used
- Python
- Random module
- String module
---
## ▶️ How It Works
The program asks the user for:
- Password length
- Include letters (y/n)
- Include numbers (y/n)
- Include symbols (y/n)
Then it generates a secure random password using selected options.
---
## ▶️ How to Run
Run the Python file:
python password_generator.py
---

## ⚠️ Error Handling
- Shows message if no character type is selected
- Ensures valid password generation only when inputs are correct
